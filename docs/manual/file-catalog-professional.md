# Professional File Catalog

This catalog provides one-line responsibility notes for each current workspace file.

## Root Files

- `.env.local` — Local environment variables for Appwrite and app runtime.
- `.gitignore` — Git ignore patterns for dependencies, build output, and local-only files.
- `README.md` — Primary project documentation and high-level technical overview.
- `components.json` — shadcn/ui generator configuration and component aliases.
- `eslint.config.mjs` — ESLint setup and code-quality linting rules.
- `next-env.d.ts` — Next.js TypeScript ambient declarations generated by framework tooling.
- `next.config.ts` — Next.js application and build configuration.
- `package-lock.json` — Locked dependency graph for reproducible installs.
- `package.json` — Scripts, dependencies, and project metadata.
- `postcss.config.mjs` — PostCSS pipeline configuration used by Tailwind.
- `tailwind.config.ts` — Tailwind theme/content scanning configuration.
- `tsconfig.json` — TypeScript compiler options and path alias config.
